Definiciones de CSS

1. ¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación de un documento escrito en HTML o XML. Se encarga de controlar el diseño visual de una página web.

2. ¿Cuál es la función principal de CSS?

La función principal de CSS es separar la presentación visual del contenido de una página web. Esto permite estilizar elementos como colores, tipografías y diseños sin modificar el contenido HTML.

3. ¿Cómo se puede incluir CSS en un documento HTML?

CSS se puede incluir de tres formas: en línea, interno y externo. La opción recomendada es el CSS externo, enlazando un archivo .css desde la sección <head> del documento HTML.

4. ¿Qué son los selectores en CSS?

Los selectores en CSS son patrones que se usan para seleccionar elementos HTML a los que aplicar estilos. Ejemplos incluyen selectores de tipo, clase y ID.

5. ¿Qué son las propiedades y valores en CSS?

Las propiedades definen qué aspecto estilístico se quiere modificar, como color o margen, y los valores especifican cómo se modifica. Ejemplo: color: blue;.

6. ¿Cómo se aplica el modelo de caja en CSS?

El modelo de caja en CSS describe cómo se calculan las dimensiones y el espacio de un elemento usando sus áreas de contenido, padding, border y margin.

7. ¿Qué es el "cascading" en CSS?

El "cascading" en CSS es el proceso mediante el cual se priorizan y aplican los estilos según el origen, la especificidad de los selectores y el orden en que se definen.

8. ¿Qué son las media queries?

Las media queries permiten aplicar estilos específicos en función de las características del dispositivo, como el ancho de la pantalla, para lograr un diseño responsivo.

9. ¿Cuáles son algunos estilos comunes que se pueden aplicar?

Los estilos comunes incluyen color, background-color, font-size, margin, y padding, entre otros.

10. ¿Cómo se puede usar CSS para crear un diseño responsivo?

Para crear un diseño responsivo se pueden usar media queries, unidades relativas como em y rem, y técnicas como Flexbox o Grid Layout.
